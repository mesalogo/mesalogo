# 资源关系图功能设计

## 概述

在首页增加交互式资源关系拓扑图，直观展示系统内各资源的关联关系，并提供快捷操作入口。

## 实现步骤

1. **后端 API**
   - 新增 `/api/statistics/resource-graph` 接口
   - 查询各资源表及关联表，构建节点和边数据

2. **前端组件**
   - 创建 `ResourceGraph` 组件
   - 使用 ECharts graph 图表
   - 实现节点点击、悬停、拖拽交互
   - 实现筛选和视图切换功能

3. **首页集成**
   - 在 Home.tsx 中新增资源关系图卡片
   - 放置在核心指标和活动趋势之间

## 技术选型

- **图表库**: ECharts (已在项目中使用)
- **图类型**: graph (力导向图)
- **布局算法**: force (力导向布局)
