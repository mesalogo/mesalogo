## 附录：设计规范备注

### 字体建议
- 中文标题：思源黑体 Bold / 阿里巴巴普惠体 Bold
- 中文正文：思源黑体 Regular
- 英文/数据：Inter / SF Pro Display
- 金句/引用：楷体 / 宋体（增加文化感）
- 大数字：DIN Alternate / Bebas Neue

### 配色方案
- 主背景：#0D1117（深色）或 #1A1A2E
- 主文字：#FFFFFF
- 品牌高亮：#00D4AA（青绿）
- 辅助色：#4A90D9（蓝）、#FF6B6B（红，用于警示数据）、#FFD93D（黄，用于强调）
- 引用/来源：#8B949E（灰色）

### 图表风格
- 极简线条，避免立体效果
- 数据可视化用品牌色系
- 图标用线性风格（Lucide / Phosphor / Tabler Icons）
