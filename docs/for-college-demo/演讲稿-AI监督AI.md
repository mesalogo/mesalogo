## 第二幕：双引擎规则 —— 给 AI 行为立"国标"（约 5 分钟）

`[切片 → 双引擎标题页]`

### 2.1 为什么需要两个引擎？

刚才我提到监督者检查规则时用了"双引擎"。这个词不是噱头，它解决的是一个真实的技术矛盾。

`[切片 → 两种规则对比表]`

教育场景中有两类规则需求：

**一类是模糊的、需要语义理解的**。比如——
> "当学生表现出明显的挫败感时，AI 应该给予鼓励，而不是继续追问。"

这种规则，你没法写成代码。因为"挫败感"是一种需要语境理解的判断。这交给大模型来做。我们叫它[强调]**自然语言规则引擎**。

**另一类是精确的、需要确定性的**。比如——
> "如果学生连续答错 3 次以上，自动降低题目难度。"

这种规则必须 100% 确定性执行。不能让大模型"差不多"判断。我们用 JavaScript 在安全沙盒中执行。沙盒有严格的约束——5 秒超时、50MB 内存上限、完全隔离。我们叫它[强调]**逻辑规则引擎**。

`[切片 → ISO标准类比]`

做标准化的同学会觉得很熟悉——

自然语言规则 → 就像 **ISO 管理体系标准**，比如 ISO 9001，它写的是原则和方向。

逻辑规则 → 就像 **ISO 技术标准**，比如 ISO 17025，它写的是精确的检测方法和数值要求。

`[手势]` 一个管"方向"，一个管"精度"。两条腿走路。

`[缓]` 这就是我们产品名字的由来——**双引擎规则系统**。也是我们区别于市面上所有 AI 教育产品的核心技术壁垒。

`[停顿]`

### 2.2 教学场景演示

`[切片 → 标准化课程场景]`

让我用一个具体的教学场景来说明。

假设这是一节标准化工程课。主题是"国际标准草案审议模拟"。

教师在系统中做了两件事：

`[切片 → 规则编写界面]`

**第一，写入自然语言规则：**
> "在讨论国际标准时，必须考虑发展中国家的技术转移能力。任何忽视发展中国家利益的提案，应被标记并提醒。"

这条规则由大模型在每次对话中实时理解和判断。

**第二，写入逻辑规则：**
```javascript
if (proposed_tolerance < 0.001 && country.development_level === 'developing') {
  flag("该公差要求可能超出发展中国家现有检测设备能力，建议设定过渡期");
}
```

这条规则在沙盒中精确执行，不存在"大概""可能"。

`[切片 → 讨论画面]`

然后，6 个 AI 角色开始讨论——中国代表、德国代表、美国代表、产业界代表、实验室代表、发展中国家代表。

在第三轮讨论中，德国代表提出了一个超高精度公差要求。监督者智能体立即触发逻辑规则检查——检测到公差值 0.0005 小于阈值 0.001，且涉及发展中国家。规则触发。

`[手势]` 监督者介入：*"请注意，该提案的公差要求可能超出发展中国家现有检测设备能力。建议在标准文本中增加过渡期条款。"*

`[停顿]`

学生亲眼看到——**一条规则如何在多方博弈中真实地影响了讨论走向**。

然后教师说："好，现在你们来修改这条规则，把阈值从 0.001 改成 0.01，看看会发生什么。"

`[缓]` 学生修改规则 → 重新运行讨论 → 观察 AI 行为变化 → 理解[强调]**"标准如何塑造行为"**。

这才是标准化工程教育的灵魂——不是背条文，而是[强调]**亲手编写规则，观察规则如何驱动复杂系统运转**。

## 附录：Q&A 预备问题

**Q1：监督者智能体会不会也出错？**
> 监督者本身也是 AI，确实可能出错。但它执行的逻辑规则部分是 100% 确定性的（代码沙盒执行），只有自然语言规则部分存在 AI 判断的模糊性。我们的设计理念是：用确定性规则兜底关键红线，用语义规则处理灵活判断。同时，所有规则检查结果都有完整日志，教师可以事后审查和修正。

**Q2：这套系统对教师的技术要求高吗？**
> 自然语言规则完全是用日常语言编写的，不需要任何编程知识。逻辑规则需要简单的 JavaScript 语法，但我们也提供模板库，教师可以直接选择和修改。未来我们也在探索用 AI 辅助教师生成规则——教师只需描述教学意图，系统自动生成对应的双引擎规则。

**Q3：跟 Dify/Coze 这些平台有什么区别？**
> Dify、Coze 是通用的 AI 应用构建平台，侧重于工作流编排和 API 集成。我们的核心差异在三个方面：一是监督者智能体（它们没有），二是双引擎规则系统（它们只有 Prompt 层面的约束），三是平行实验室（它们没有对照实验能力）。此外，我们的环境变量系统支持模板-实例架构，专为仿真推演设计。

**Q4：如何与"一带一路"标准化教育联盟结合？**
> 中国计量大学是"一带一路"标准化教育与研究大学联盟的牵头单位。这套平台可以为联盟高校提供统一的标准化教育仿真环境——不同国家的高校可以在同一个行动空间中模拟国际标准制定过程，这对于培养具有全球视野的标准化人才具有重要意义。
