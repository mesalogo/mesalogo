### 3. 知识库绑定逻辑（关键修复）✅

**修复前**:
```javascript
// 问题：某个知识库绑定失败，抛出错误，后续知识库不会被处理
for (const kbId of internalToBindIds) {
  await knowledgeAPI.mountToRole(roleId, kbId);  // 失败直接抛出
}
```

**修复后**:
```javascript
// 正确：每个知识库独立处理，单个失败不影响其他
let hasError = false;
for (const kbId of internalToBindIds) {
  try {
    await knowledgeAPI.mountToRole(roleId, kbId);
  } catch (error) {
    console.error(`内部知识库绑定失败: ${kbId}`, error);
    hasError = true;
    // 不抛出，继续处理下一个
  }
}
// 最后统一提示
if (hasError) {
  message.warning('部分知识库操作失败');
}
```

**验证结果**: ✅ 修复正确，行为与原文件一致
